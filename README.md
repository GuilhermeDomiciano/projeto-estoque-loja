# Projeto Estoque Loja

Este projeto tem como objetivo fornecer um sistema de gerenciamento de estoque para lojas, facilitando o controle de produtos, fornecedores e vendas. Desenvolvido por Guilherme Domiciano e Geisbelly Feitosa.

## Sobre o Projeto

O sistema permite que os usuários:

- Cadastrem e gerenciem produtos.
- Registrem fornecedores.
- Acompanhem o histórico de vendas.
- Gerem relatórios de estoque.

O objetivo é otimizar o processo de controle de estoque, reduzindo erros e aumentando a eficiência operacional das lojas.
